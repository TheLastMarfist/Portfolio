Juno

Juno is a project focused on making Attack & Defense style cybersecurity competitions more widespread. The way it will fix this is by making it easier to spin up infrastructure for these competitions. When setup correctly, Juno will create virtual machines and networks for people to attack and defend with only high level input from the user about what types of vulnerabilities and systems they want.

Catapult

Catapult is an automated exploit thrower, designed for use at DEFCON CTF. This is a web app that allows users to submit exploits and mark what teams they should be run against. Catapult will then automatically run these exploits against the given teams every game round and if a flag is captured will also submit it. Catapult was written in python.

CTF League 2021-2022

CTF League is school year long competition put on by members of the OSU Secuirty Club (OSUSEC) for other members. Each week a new challenge is created by the coaches and the participants split into random teams to try and solve the challenge. I participated in the league from 2021-2022 and achieved first place.
